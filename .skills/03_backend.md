# Skill: Backend Rules
- **FastAPI**: Use async-first endpoints.
- **SQLAlchemy**: Use async sessions.
- **Repository Pattern**: Abstract database access.
- **Dependency Injection**: Utilize FastAPI's Depends().
- **Pydantic**: Use for all validation and serialization.
- **Alembic**: For all database migrations.
- **Error Handling**: Standardized HTTP exception handlers.
- **Logging**: Use structured JSON logging.
