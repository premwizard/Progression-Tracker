# Skill: Project Overview
## Purpose
Explain the project vision, objectives, scope, tech stack, folder structure, and coding philosophy.

## Vision
Progression Tracker is an AI-Powered Goal Planning, Progress Tracking & Productivity Intelligence Platform.

## Tech Stack
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: FastAPI, Python, SQLAlchemy, PostgreSQL, Redis, Alembic
- **AI**: LangGraph, PydanticAI, MCP, Qdrant
- **Mobile**: Flutter
- **Infrastructure**: Docker, GitHub Actions, Vercel, Render, Cloudflare, AWS
