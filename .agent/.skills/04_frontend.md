# Skill: Frontend Rules
- **Next.js App Router**: Use server components by default, client components only when interactivity is needed.
- **State Management**: TanStack Query for server state, Zustand for global client state.
- **Styling**: Tailwind CSS + shadcn/ui.
- **Accessibility**: Ensure ARIA standards are met.
- **Performance**: Optimize images, fonts, and use dynamic imports.
